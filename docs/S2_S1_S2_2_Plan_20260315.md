## 二、问题根因

### 2.1 当前错误调用逻辑 ❌

**文件**: `runners.py` L55-74

```python
# 当前错误实现 ❌
if path_search_enabled:
    # BUG 1: 直接跳过 run_retro_scan()
    dipolar_intermediate = product_xyz_file  # ← 错误: 应该是中间体!
    
    # BUG 2: start 和 end 都传了产物
    hunter.s2_engine.run_path_search(
        start_xyz=dipolar_intermediate,   # ← product_xyz_file!
        end_xyz=product_xyz_file,       # ← product_xyz_file!
        ...
    )
```

**正确逻辑应该是**:
```python
# Step 1: S2.1 拉伸搜索 (生成偶极子中间体)
(intermediate_xyz, ...) = run_retro_scan(product_xyz, ...)

# Step 2: S2.2 路径搜索 (使用中间体作为起点)
if path_search_enabled:
    (ts_guess_xyz, ...) = run_path_search(
        start_xyz=intermediate_xyz,    # ← 正确: 中间体
        end_xyz=product_xyz_file,      # ← 正确: 产物
        ...
    )
```

```python
# 旧逻辑 (错误)
```
