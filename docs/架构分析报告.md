一、系统定位与架构总览
1.1 系统定位
Exclean 是一个模块化化学文献数据提取系统，解决的核心问题是：将非结构化的科学文献（PDF）和半结构化的数据库导出（Reaxys Excel）转化为机器可读的结构化反应数据。
覆盖的反应类型：
反应族
[5+2] 氧化吡啶鎓环加成
[4+3] 丙二烯胺环加成
[4+2] Diels-Alder
衍生化反应

5.2 核心算法
SMARTS 前体分类 (Step 4 前置)
PRECURSOR_SMARTS = {
    'oxidopyrylium': {
        'type_A': "[#6]C(=O)O[C;H1,H0]1[O][C;H2]C(=O)C=C1",   # 标准 OAc
        'type_B': "[#6]C(=O)O[C;H1,H0]1[O][C;!H2]C(=O)C=C1",  # C6 取代
        'type_C': "[OH][C;H1,H0]1[O][C]C(=O)C=C1",             # 半缩醛
    },
    'allenamide': {
        'general': ['[CH2]=[C]=[C][NX3]', '[C]#[C][CH2][NX3]'],
    },
}
关键: 必须使用 sanitize=False + UpdatePropertyCache(strict=False)
原子映射 + 健全性检查 (Step 3)
1. RXNMapper.get_attention_guided_atom_maps()
2. 置信度阈值: ≥ 0.8 → OK, < 0.8 → LOW_CONFIDENCE
3. 健全性验证:
   - 原子映射重叠 ≥ 4 个原子
   - 重叠比率 ≥ 0.4
   → 否则: FAIL_INSUFFICIENT_MAP
4. SQLite WAL 缓存 (避免重复计算)
反应核心提取 (Step 4)
1. 解析映射反应 SMILES
2. 根据前体类型选择 SMARTS 模式组
3. 匹配候选原子集
4. 推断键变化 (formed/broken)
5. 计算成环尺寸:
   - 双键移除法: 移除新形成键 → 最短路径
   - SSSR 环并集法: 融合环周长
6. 领域知识修正: 呋喃体系氧桥捷径 (计算5/6 → 修正为7)
3 层产率提取 (Step 6)
层级
1
2
3
特殊处理:
- nr/no reaction → yield=0.0, censored
- trace → UNCERTAIN
- 区间 70-80% → 中点 (0.75)
- 上界 <70% → 标记 is_upper_bound
质量平衡验证 (Step 9)
1. 计算分子式: rdkit.Chem.Descriptors.CalcMolFormula()
2. 逐元素减法: Reactant - Product = Delta
3. 匹配离去基团:
   C2H4O2 → AcOH (乙酸)
   C3H10OSi → TMSOH (三甲基硅醇)
   H2O → H2O
4. 负 Delta 处理:
   - 氧化加氧 → 检查氧化剂关键词
   - [4+3] + DMDO → NOT_EVALUABLE

score = 1.0

  -0.20  产率异常, -0.10 产率缺失
